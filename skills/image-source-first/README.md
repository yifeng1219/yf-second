# image-source-first

一个给图片请求用的流程型 skill。

## 作用

默认执行这套顺序：

1. 先搜索现成可用图片
2. 找到后先做发送前质检
3. 没问题直接发给用户
4. 如果搜不到或素材不合格，再生成或重做
5. 生成后仍然先检查，再发送

## 适用场景

- logo / 图标 / 品牌素材
- 透明 PNG
- 官方公开图
- 参考图里的单独元素
- 用户明确说“先搜现成图”

## 核心规则

- 优先搜索，不先盲目生成
- 优先附件直发，不只丢链接
- 发送前必须检查文件存在、可读、尺寸正常、内容正确
- 任一检查不通过，直接打回重做
