# zt-image-copywriter

这个 skill 专门解决：图能做出来，但文案层级不够强、不够顺、不够像成品广告。

## 核心目标

把原始文案整理成适合上图的版本，让图片更像专业海报、宣传图或封面。

默认优化方向：
- 标题更短更狠
- 副标题更清楚
- 卖点更像人话，不堆词
- CTA 更直接
- 结构更适合上图排版

## 适用任务

- 海报标题优化
- 宣传图卖点重写
- 封面标题改写
- 文案层级梳理
- 广告图按钮文案优化
- 把参考图里的原文案整理成新版结构

## 标准流程

1. 先判断图片类型
   - 广告海报
   - 服务宣传图
   - 社媒封面
   - 课程/活动介绍图

2. 提取原始信息
   - 主主题是什么
   - 最强卖点是什么
   - 用户最该先看到哪句话
   - 哪些字该缩短，哪些字该上移到标题

3. 重组文案层级
   默认拆成：
   - 主标题
   - 副标题/解释句
   - 3~5 条卖点
   - CTA / 按钮文案
   - 角标/补充说明（可选）

4. 重写时遵守这些规则
   - 标题不要太长
   - 一屏内优先短句
   - 卖点不要重复表达
   - 不要堆空泛大词
   - 先讲结果，再讲过程
   - 尽量符合中文广告阅读习惯

5. 输出时给可直接上图版本
   - 默认直接给一版成品文案结构
   - 如有必要，再补 2~3 个标题备选
   - 不要只说“可以优化成更高级”，而不给具体文案

## 常用结构模板

### 模板 A：服务宣传图
- 主标题：解决核心痛点
- 副标题：说明服务方式或适用对象
- 卖点：3 条内，短句
- CTA：立即咨询 / 远程处理 / 专业协助

### 模板 B：小红书/社媒封面
- 主标题：强对比、强结果
- 副标题：补场景或对象
- 角标：限时 / 干货 / 教程 / 避坑 / 实测

### 模板 C：门店/转化型广告
- 主标题：服务名 + 结果
- 卖点：直给
- CTA：马上咨询 / 一对一处理 / 在线解决

## 输出格式

默认按这个格式给：

- 主标题：
- 副标题：
- 卖点1：
- 卖点2：
- 卖点3：
- CTA：

如果用户要更多选择，再补：
- 标题备选 A
- 标题备选 B
- 标题备选 C

## 示例调用语

- 用 zt-image-copywriter，帮我把这张海报文案优化一下
- 调用 zt-image-copywriter，把标题改得更有广告感
- 用 zt-image-copywriter，把这张图的卖点重新分层
- 用 zt-image-copywriter，给这张宣传图出一版可直接上图的文案
- 用 zt-image-copywriter，把按钮文案和副标题一起重写

## 与其他 skill 的配合

- 先用 `zt-image-copywriter` 梳理文案，再用 `zt-image-suite` 作图
- 遇到参考图重做时，可先用 `zt-image-copywriter` 重写信息层级，再交给 `zt-poster-redesign`

## 禁忌

- 不要只给原则，不给成品文案
- 不要标题又长又虚
- 不要卖点全是套话
- 不要写成大段文章，必须适合上图
